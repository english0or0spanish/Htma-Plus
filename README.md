# Htma+
Better HTML/HTA

STILL IN **ALPHA** PHASE MAY BE BUGGY OR FEATURES MAY NOT WORK!

## Features:
- JavaScript plugins
- place `<display value=0>`  inside your .HTMA file and it will disable the built in display module

## Requirements:
- Python 3.13+

## Compatibility:
- This project was built to run on **Windows**

## Installation:
1. Click on the green button that says 'Code'
2. Click on 'Download ZIP'
3. Extract the ZIP file to the folder of your choice
4. Run **Patch.bat**
5. Complete
